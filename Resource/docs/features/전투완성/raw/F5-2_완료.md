## 1. 무엇이 됐나 — F5-1 이월 3건 해소 + 스킵 즉시화

F5-1은 "한 팀 전멸 시 전투 종료"까지만 해결했고, 죽은 유닛이 계속 턴을 받아 공격하고·시체가 클릭되고·쓰러짐 연출이 없는 3개 갭이 남아 있었다([[F5-1_완료]] §3). F5-2가 이 3건을 **3청크 + 스킵 즉시화**로 해소했다.

### 청크1 — `EnterTurnStart` TS1~TS6 (사망/STUN 턴스킵 + 쿨다운 스윕 + 막기 해제)
[[F5_착수지시서]] H-1/H-2가 지시한 파이프라인을 배선.
- 사망(TS1)·STUN(TS4/TS5) 턴스킵 + 쿨다운 스윕(TS3 −1, 하한0) + 막기 해제(bBlockActive).
- **H-1 준수**: TS1(사망 스킵 Branch)이 기존 `SetTurnCounter(+1)`보다 **앞** — 사망 스킵이 TurnCounter를 올리지 않음(광폭화_재검증 §2·F9a 원장 보호).
- **H-2 준수**: `EnterTurnEnd`가 이제 진입점 3개(정상·TS1·TS5)의 **3-way merge**이며, 스킵 경로는 `EnterTurnEnd` 호출 후 **exec 종단**(then 미연결 — 큐 2칸 점프·시체 커맨드 방지).
- **Director 노드 검증**: H-1(순서)·H-2(종단) 정적 확인.

### 청크2 — `PlayHurtReaction` DYING 분기 (쓰러져 고정)
[[F5_착수지시서]] B5 순서를 배선: `Row13/FC5 → RetriggerableDelay(0.575) → bFreeze=1`(idle 복귀 없음 — 쓰러지는 동작이 재생된 뒤 마지막 프레임에 고정).
- **declaring_class=MID 명시 4개**: `SetScalarParameterValue`의 `declaring_class=/Script/Engine.MaterialInstanceDynamic` 명시(노하우 §L130 — 미명시 시 MPC 오버로드로 무음 실패=시체 꿈틀거림).
- **G-A 재동결 가드**: 내곽 `Branch(bAlive)`로 시체 재타격 시 DYING 재재생 방지.
- **ClickBox NoCollision(최상단)**: 사망 시 `ClickBox`를 NoCollision으로 전환해 시체 타겟 불가(하이라이트는 F5-1에서 이미 사망자 제외였고, 여기서 클릭 자체를 차단).
- **Director 검증**.

### 청크3 — `ResetForBattle()` 신규 + `InitBattle` 리셋 확장 + `EnterEnd` HighlightOff
[[F5_착수지시서]] B3(a안: 완전 재시작) 배선.
- 유닛 `ResetForBattle()` 신설: `SetScalar` 4개(MID) · `Hp=MaxHp`(G-C — Hp는 BeginPlay에서만 로드되던 것을 리셋 시 복원) · `IsValid` 가드(G-G).
- `InitBattle` 리셋 확장 + **기존 `CurrentIndex`/`TurnCounter` 리셋 보존(G-H)** — 신규 리셋을 얹으면서 기존 리셋을 덮어쓰지 않았음을 확인.
- `EnterEnd`에 HighlightOff 추가.
- **Director 검증**.

### 스킵 즉시화
`bWasSkip` 플래그로 스킵 턴은 `EnterTurnEnd`의 `Delay(0.35)`를 우회해 `Delay(0.0)`로 처리 — 죽은/기절 유닛의 턴이 0.35초씩 끌지 않고 즉시 넘어간다. **실제(살아있는) 턴의 페이싱은 불변.**

## 2. 검증 — 오너 실플레이 + 스모크

**오너 실플레이 검증(PIE)**: 아래 전부 확인.
- 죽은 유닛 턴스킵(더 이상 시체가 공격 안 함)
- 시체 클릭 불가
- 쓰러져 고정(DYING 재생 후 마지막 프레임 정지, 꿈틀거림 없음)
- 스킵 즉시화(죽은 유닛 턴이 즉시 넘어감)

**`InitBattle` → `ResetForBattle` 스모크**: `Accessed None` 0건 + `Init → TurnStart → AwaitCommand` 정상 진행 확인.

## 3. 이월

- **#4 End 버튼 재전투**: `ResetForBattle`을 End 화면 버튼에 배선하는 작업 — 별도 작업으로 이월(로직은 완비, UI 배선만 남음).
- **orphan 노드 정리**: 무해한 고아 노드 잔존 — 정리 이월(기능 영향 없음).
- **`bWasSkip` "Is Not Valid" 초예외 경로**: 이전 값을 물려받음 — 정상 플레이에는 무해(초기 극단 예외 경로에서만 발생).
