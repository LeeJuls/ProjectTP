# 청사진 — 걸어나오기연출

옥토패스식 "나와서 치고 돌아오기" + 오너 지정 공격 포인트. 턴제전투MVP Executing 예약 지점에 삽입.
#projectTP/걸어나오기연출

## 목표
- 자기 턴에 유닛이 앞으로 나와(WalkForward) 공격 후 제자리로 복귀(WalkBack).
- 공격 전 멈춤 포인트는 적/아군 각각 수동 지정 가능(기본은 Director/시스템 설정, 오너가 레벨에서 드래그 조정).

## 설계 확정판 요약 (승인 plan 전문은 `plan.md` 참조)
- 공격 포인트: `BP_AttackPoint`(에디터 표식) 2기 + 유닛 인스턴스별 `AttackPointOverride`(최우선) + 폴백(HomeLocation±300).
- 3단 우선순위: ①AttackPointOverride ②팀 포인트(Manager 경유) ③폴백.
- z-fight 자동 가드: 도착 Y가 대기 중인 다른 유닛과 10cm 이내면 12cm 비킴.
- 연출 시퀀스: 잠금→마커OFF→WalkForward()(즉시반환, latent 내부 0.4s)→Delay(0.55)→PlayAttack→Sequence[Delay(0.25)→TakeHit / Delay(0.75)→WalkBack()→Delay(0.45)→TurnEnd].
- W1 범위: BP_AttackPoint 신규 + 배치, 유닛 확장(3단 결정+Y가드+WalkForward/WalkBack+이펙트 재계산), RUN1 스팟체크.
- W2 범위(별도 작업): Manager Executing 재배선(마커OFF 신규 삽입 + WalkForward/WalkBack 호출 삽입).
