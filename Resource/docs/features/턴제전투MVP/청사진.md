## 목표
배치_1(옥토패스대치 8기 정면대치) 위에서 **핫시트 "공격→피격" 루프**를 완성한다. 턴이 캐릭터 하나씩 순환하고, 자기 턴에 Attack 버튼 → 적 타겟 클릭 → 공격 모션 → 피격 모션 → 다음 턴으로 넘어가는 4v4 전원 수동 조작(AI 없음) MVP.

## 범위 / 비범위
- ✅ 범위: 턴 순환(무한), 클릭 기반 타겟 선택, 공격/피격 애니 싱크, 턴 마커·발광 하이라이트 UI, 취소·잠금 등 엣지 처리, PIE 재시작 리셋.
- ❌ 비범위: 데미지·HP·사망·승패 판정, 걸어나오기 등 이동 연출(Executing 삽입 지점만 예약), ATB 게이지(TurnStart 선택 로직 교체 지점만 예약), AI, 무효클릭 피드백·발광 펄스·장기 float 안정성(관찰만, 이월).

## 큰 흐름
1. **Init**: 전 유닛(8기) self-register 완료 게이트 후 큐 빌드 `[A1,B1,A2,B2,A3,B3,A4,B4]`, 초기화(발광0·마커숨김·잠금).
2. **TurnStart→AwaitCommand**: 큐의 다음 유닛에 마커 ON, 입력 잠금 해제, "Attack" 버튼 노출.
3. **AwaitTarget**: 버튼 클릭 시 상대팀 발광 ON, 라벨 "Cancel"로 전환. 유효 타겟(상대팀) 클릭 시 Executing 진입, 버튼 재클릭 시 취소(AwaitCommand 복귀).
4. **Executing**: 입력 즉시 잠금, 공격 모션(0.70s) 재생 중 t=0.25에 피격 모션(0.45s) 트리거. t=0.75에 TurnEnd.
5. **TurnEnd**: 마커 OFF → 인덱스 증가(modulo 실길이) → 0.35s 텀 → 다음 TurnStart로 무한 순환.
