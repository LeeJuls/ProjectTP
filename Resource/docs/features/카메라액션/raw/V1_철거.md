## 사전 확인 — plan이 지목한 dangling 노드 부재
plan(`humble-purring-glacier.md`)이 V0 항목으로 지목한 `K2Node_CallFunction_206`(FindLookAtRotation)·`_207`(BreakVector)를 `find_nodes`로 EventGraph 전체 조회했으나 **실그래프에 존재하지 않았다**(59개 노드 전수 확인, `_206`/`_207` 부재). 이는 plan 검토 세션(GE 사전검토) 시점과 이 실행 세션 시작 시점 사이에 이미 정리됐거나 별도 스크래치 그래프였던 것으로 추정 — 존재하지 않는 노드 삭제는 무의미하므로 이 항목은 "확인 결과 부재 — 스킵"으로 처리하고 V1 본 작업(EnterExecuting/TurnEnd 블록 철거)으로 진행.
