# 🎮 캐릭터 배치 가이드 (오너용)

> 레벨 `map_battle_octopath`에 **배치 포인트 8개**(BP_BattleSpawnPoint)가 준비됨. 에디터에서 자유롭게 드래그하세요. 8개 각각 독립 수정 가능.
#projectTP/옥토패스대치

## ✅ 하는 법
| 작업 | 방법 |
|---|---|
| **포인트 선택** | 아웃라이너 → `BattleStage/SpawnPoints` 폴더 → 이름 클릭 (`SpawnPoint_Party_A1`~`Enemy_B4`) → **F 키**로 카메라 포커스 |
| **이동** | **W**(이동 기즈모) 상태에서 드래그 — 화면에서 보이는 그대로 움직임(WYSIWYG) |
| **바닥에 붙이기** | 선택 후 **End 키** |
| **방향 전환** | Details 패널 → **Face Left** 체크박스 (☑=왼쪽 봄/아군, ☐=오른쪽 봄/적) — 빨간 화살표가 방향 표시(게임에선 안 보임) |
| **전투 카메라 시점으로 보기** | 아웃라이너에서 `CameraActor_0`(BattleCamera) 우클릭 → **Pilot** (해제: 뷰포트 좌상단 Pilot 정지 버튼) |
| **저장** | **Ctrl+S** |

💡 팁: 작업 시작 때 Pilot로 전투 시점을 본 뒤 **Ctrl+1**로 북마크 저장 → 이후 **1 키**로 그 시점 즉시 복귀.

## ❌ 하지 말 것
- **회전(E)·스케일(R) 기즈모 금지** — 스프라이트가 옆면으로 사라지거나 왜곡됩니다. **이동(W)만.**
- **포인트 삭제·복제(Alt+드래그) 금지** — 8개 고정. 실수로 지웠으면 아래 "복구".
- **컴포넌트 머티리얼 직접 변경 금지** — 방향은 Face Left 체크박스만.
- Details의 X/Y 숫자 직접 수정은 비추천(카메라 각도상 좌우가 화면과 반대) — **화면 드래그가 안전.**
- 다른 카메라(`Camera`)는 건드리지 마세요. BattleCamera(`CameraActor_0`)가 전투 시점.
- 카메라 앞 `UI_AttackButton`(공격 데모용 버튼)은 건드리지 마세요 — 이동/삭제 금지.

## ⚠ 줄무늬(z-fight) 예방 — 딱 하나만 기억
**두 캐릭터를 정확히 같은 깊이(앞뒤 위치)에 겹쳐 세우지 마세요.** 조금씩(10cm+) 앞뒤로 어긋나게. 
- 에디터 상단 그리드 스냅을 **10** 으로 켜두면 실수 확률↓
- **아군 4기 배치 끝나면 한 번 불러주세요** (1차 자동검증) → 적 4기 후 2차. 마지막에 전체 검증해드립니다.

## 🔧 복구
- 방금 실수 → **Ctrl+Z** (방향이 꼬이면 Face Left를 다시 클릭하는 게 더 확실)
- 뭔가 망가짐/지워짐 → **"리셋해줘"** 라고만 하시면 초기 배치([[raw/S2p_초기배치백업]])로 복원해드립니다.
