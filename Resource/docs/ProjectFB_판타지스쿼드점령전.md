## 게임 정체성
- **3인칭 4:4** (총 8명) 판타지 스쿼드 PvP — 활(원거리) · 마법(스크롤) · 검(근접)
- 좁은 아레나 맵 (작은 마을 2개 + 중앙), **Steam**, **UE 5.8**
- **유저 자산 = 코스튬뿐** (전투 영향 0, no pay-to-win)
- **전투 재료는 매판 인게임 파밍** (배틀그라운드식) — 영속 소유 없음
- **재미 = 순수 승리** (LOL식) — 수집 도파민이 아니라 경쟁·숙련
- 차별화: 현대전 스쿼드 슈터는 많지만 **판타지 무기 3종 스쿼드 대전은 시장 공백** ("엘더스크롤 4:4" 느낌)
- **출시 = 마검전 1모드 집중** (인디 PvP 매칭 풀 보호 + 관전성 — 근거: 설계로그)

## 설계 원칙 (모든 룰의 상위 기준)
1. **심플 is best**: 배워야 할 룰 2문장 이내. 나머지는 관찰 가능 / 순간 고지 / 심판(서버) 룰.
2. **자연 학습**: 보이는 원인 → 즉각 결과 → 올바른 교훈 루프. 루프 없는 룰이 진짜 복잡도.
3. **판돈 증폭기**: 경제·드라마가 마검 하나로 수렴 — 교전과 오브젝티브의 융합.
4. **옵트인 리스크**: 큰 손실은 선택(마검을 드는 것)의 결과여야 한다.
5. **수비 이점은 스탯이 아니라 기하학** (리스폰 거리·지형).
6. **깊이는 룰이 아니라 피지컬·감각 층에서** (에임·발소리·판단 — 다음 설계 단계).

## 온보딩 — 배워야 할 룰은 이 2문장이 전부
1. **마을을 털어 강해져라.**
2. **마검을 뽑아 적 성소에 꽂으면 승리.**

## 룰 v1.0

### 매치 구조
- 4:4. 블루/레드 각자 마을(진영)에서 시작. 맵 중앙 **봉인석**에 마검이 꽂혀 있음.
- 매치 제한시간 **10분**(튜닝).
- **승리: 마검을 적 성소(수호석)에 꽂기 = 즉시 승리** (1골 결승, 타락 폭발 연출 "팍!").

### 파밍
- 마을 **건물 내부** 루팅(배그식): 무기 + 방어구 + 물약 + **스크롤**.
- **마법 = 스크롤.** 사전 클래스 없음 — **무기+스크롤+방어구 조합 = 직업**(emergent class).
  - 예: 메이스+힐+천옷=힐러 / 완드+파이어볼=화염법사 / 활+헤이스트+가죽=레인저 / 활+천리안=저격
- 파밍의 기회비용: 오래 털수록 중앙 마검을 상대에게 내줌.

### 마검
- **쥐거나, 버리거나, 꽂거나** — 인벤토리 수납 불가(저주: 칼집을 거부).
- 든 동안 자기 무기·스크롤 사용 불가, **마검(근접)으로만 전투** (성능 = 밸런스 노브).
- 위치(운반 중 포함) **항시 미니맵 표시**.
- 바닥 **5초** 방치 → 중앙 봉인석으로 귀환.
- 패스 = **드랍 릴레이**(버리고 아군이 5초 내 줍기). 별도 투척 패스 없음(v1).

### 드랍 (스테이크 — 드랍은 마검 문맥에서만 발생)
- **일반 사망: 드랍 없음.**
- 마검 **소지자가 죽으면**: 장비 1개 **확정** 드랍 (+마검 드랍).
- 마검 **소지자가 죽이면**: 희생자 장비 1개 **50%** 드랍.
- **스틸(뺏김) = 상대가 주워 드는 순간** — 바닥에 떨어진 것만으론 아님.

### 사망·리스폰
- 즉시 본진 리스폰 + **5초 무적**.
- 수비 리스폰 지점 ↔ 골(성소)은 **분리하되 너무 멀지 않게** — 이 거리가 곧 수비 이점(플랜팅 방식과 세트로 튜닝).

### 타임아웃 & 파이널 타임 — "일발 역전 드라이브"
- **0:00 판정**:
  1. 검이 정중앙(중립) → **골든 마검 서든데스**
  2. 검이 있는 진영 팀 = **패배**
  3. **단, 그 팀이 검을 쥐고 있으면 → 파이널 타임** (지는 팀 전용 마지막 드라이브)
- **파이널 타임**: 상한 **60~90초**(튜닝). 플랜팅 = **역전승** / 상대가 주움 = 즉시 종료 / 드랍 후 **3초** 방치 = 패배 / 만료 = 패배. 아군이 3초 내 재획득하면 드라이브 계속.
- 마지막 1분의 대본(룰이 자동 연출): 지는 팀 = 소지 확보 다이브("잡으면 연장!") / 이기는 팀 = 캐리어 사냥 — 단 너무 일찍 죽이면 5초 워프가 검을 중앙(=서든데스)으로 보내므로 막판 킬 타이밍이 미세 게임.

## 연출 요구사항 (룰 아님 — 자연학습 지원, 구현 시 필수)
1. **봉인석 귀환 카운트다운 가시화**: 떨어진 검이 5초간 어둠에 잠겨가는 연출 (안 보이면 "가끔 사라짐"으로 오학습)
2. **탐식 VFX**: 소지자 킬마다 검이 빨아먹는 이펙트(확률 무관 항상), 전리품 팝은 50%일 때만
3. **종료 60초 전**: 경고 배너 + 검 고동("쥐면 마지막 드라이브") + 검 있는 진영에 저주 안개 번짐(시각 전용, 데미지 없음)
4. **마검 유혹**: 소지자 시체에서 검이 떠올라 죽인 자를 향해 기울며 속삭임 — 줍기는 여전히 선택
5. **파이널 타임**: 검신 균열 + 붉은 발광
6. **플랜팅**: 성소 타락 폭발 웨이브
- 감각 층 후보(차기 설계 대상): **마검의 속삭임**(벽 너머 근접 청각 레이더), 화살 낙차, 골목 발소리, 스크롤 영창음

## 미결 (v1.0 이후 결정)
1. **플랜팅 방식**: 터치 즉시 vs 채널링 vs 투척 골인 — 리스폰~골 거리와 커플링 튜닝
2. 드랍되는 "장비 1개"의 선정 방식 (랜덤 vs 최고가치 등)
3. 맵 규모·레이아웃 (마을 2 + 중앙 봉인석 + 성소 위치)
4. 정식 명칭 (게임: Project-FB 가칭 / 모드: 마검전 가칭)

## 서랍 (보관 — 출시 제외)
- **거점 점령(도미네이션)**: 제2모드 후보. 꺼낼 때 패치 세트 = 틱 누적제 + 3포인트 + 10~12분 + "포인트=파밍 구역 해금". 관전성 위해 **하드포인트형 변형**(활성 거점 1개 순회) 권장.
- **지역 점령전**: 보류 (구조 리스크 — 설계로그 Fable 평가 참조)
- **젬 그랩**: 미상세 후보
- **스킨 예비안**: 용의 알·왕관·운석핵·뿔피리 (이벤트/후속 모드 재활용)
- **미채택 메커닉**: 금단의 경계선·깃발의 저주·원혼의 추격 / 영혼 부활·세 개의 제단·깃발 슛·부활의 종 (필요 시 재소환)
