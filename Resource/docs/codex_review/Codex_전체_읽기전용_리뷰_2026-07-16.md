## 바로 필요한 오너 결정

1. 유료/third-party 자산은 계속 제외하되 first-party Blueprint·DataTable·UI·맵을 어떤 source-control/backup 체계에 넣을지 결정한다.
2. Start와 F7b 중 하나를 먼저 완료해 F9a 23턴 원장과 에셋 스냅샷을 봉인한다.
3. F7b의 캐스트당 BattleLog 필드·effect roll·PRE/ON/POST 표현을 실배선 전에 확정한다.
4. Claude가 `init_unreal.py`의 부분 완료·rename·임시 에셋 충돌을 수정하고 startup 실행본을 비활성화한다.
5. 생성된 인증 토큰을 회전하고 Obsidian plugin `data.json`을 Git ignore한다.
